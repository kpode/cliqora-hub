# Cliqora Hub Landing Page

Responsive static landing page built with HTML, CSS and JavaScript.

## Files
- `index.html` — all page content and sections
- `style.css` — visual design, responsive layouts and animations
- `script.js` — mobile navigation, scroll reveals and animated statistics
- `assets/` — SVG logo and icons

## Important edits before publishing
1. Replace every `https://t.me/` URL with the real Telegram invite link.
2. Replace every `https://wa.me/` URL with the real WhatsApp invite link.
3. Edit placeholder statistics and testimonials in `index.html` as required.
4. Add real Privacy Policy and Terms of Service URLs.

## Local preview
Open `index.html` in a browser, or run a local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Deploy
Commit the folder to the GitHub repository connected to Cloudflare Pages. Cloudflare should redeploy automatically after `git push`.
